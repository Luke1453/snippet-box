# SnippetBox

A project from Alex Edwards - Let's GO  
I used to learn GO web development

## Logging basics

Application by default will log:  
INFO logs -> stsout  
ERROR logs -> sdterr  

To write app logs to file run it like this:  
```$ go run ./cmd/web >>./info.log 2>>./error.log```
